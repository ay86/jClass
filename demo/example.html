jClass.js 示例 基于 jClass 1.0.13 版本

jClass.js

一个 javascript 类库，提供基本的选择器以及若干方法，可以把她理解成 jQuery，部分 api 使用方式也一致。

Learn more

选择器
  • 标签选择器 - $('body')
  • 类选择器 - $('.class')
  • ID选择器 - $('#id')
  • name选择器 - $('@name')
  • 属性选择器 - $('[type]')
  • 多级选择器 - $('#parent .children')
  • 作用域选择器 - $('div', document.body)
  • 创建对象选择器 - $('<div>create div</div>')
注意：IE里会将NAME和ID等同化！
对象查找
  • 前一个 - $(selector).prev()
  • 后一个 - $(selector).next()
  • 第一个 - $(selector).first()
  • 最后一个 - $(selector).last()
  • 索引查找 - $(selector).eq(0)
  • 下级元素查找 - $(selector).find(selector)
  • 遍历 - $(selector).each(function(index){}) function的第一个参数为遍历的索引值

对象的属性/方法
  • 添加样式 - $(selector).addClass(className)
  • 移除样式 - $(selector).removeClass(className)
  • 是否包含Class - $(selector).hasClass(className)
  • 属性操作 - $(selector).attr(name, [value])
  • 移除属性 - $(selector).removeAttr(attribute)
  • 添加子元素 - $(selector).append()
  • 作为子元素添加 - $(selector).appendTo()
  • 在之后添加 - $(selector).after()
  • 在之前添加 - $(selector).before()
  • 添加到之后 - $(selector).afterTo()
  • 添加到之前 - $(selector).beforeTo()
  • 拷贝元素 - $(selector).clone()
  • 移除元素 - $(selector).remove()
  • 透明度设置 - $(selector).alpha(value)
  • css样式值 - $(selector).css(object || attribute, [value])
  • DataSet - $(selector).data(name, [value])
  • 宽度 - $(selector).width()
  • 高度 - $(selector).height()
  • 内部宽度 - $(selector).innerWidth()
  • 内部高度 - $(selector).innerHeight()
  • 显示 - $(selector).show()
  • 隐藏 - $(selector).hide()
  • 元素内容html - $(selector).html([html])
  • 元素本身html - $(selector).outerHtml()
  • 元素的纯文本内容 - $(selector).text([text])
  • 元素的值 - $('input').val()
  • 获取实际位置 - $(selector).get()
    • .top
    • .bottom
    • .left
    • .right
  • 淡入淡出 - $(selector).fade([value])
  • 表单项序列化 - $(selector).serialize()
  • 震动效果 - $(selector).shake(...)
  • 形状动画 - $(selector).animateShape(...)
  • 选项框操作 - $(selector).checkBox('all'|'other'|'not')

事件
  • 事件绑定 - $(selector).on(event, [selector, function]) 需要扩展方法addEvent()支持
  • 链式引用 - $(selector).html(...).show()

绑定事件代码示例


// 绑定a标签的点击事件
$('a').on('click', function(){
	console.log(this);
	return false; // 阻步冒泡
});
		

// 事件代理, 绑定事件在父级上由子选择器对象触发
$('ul#abc').on('click', 'li.on', function () {
	console.log(this);
});
扩展
  • 页面加载完成 - $.load(function)
  • DOM加载完成 - $.domReady(function)
  • 绑定事件 - $.addEvent(element, trigger, function)
  • 删除事件 - $.delEvent(element, trigger, function)
  • 浏览器检测 - $.isWho()
  • 十六进制颜色值转换为RGB - $.colorToRgb(colorValue)
  • RGB转换为十六进制颜色值 - $.colorToHex(rgbValue)
  • 多选下拉框选项转移 - $.listMove(select1, select2)
  • 获取一段HTML里某个节点的HTML - $.getHTML(html, selector)
  • Cookies操作 - $.cookie
    • .get(cookie)
    • .set(cookie, value, expired)
    • .del(cookie)
  • AJAX - $.ajax(options);
